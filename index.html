Project Introduction

“Text Marks The Spot” is a geolocation-based mobile messaging application. The system serves as a mobile application geared towards both the iOS & android markets. TMTS is sponsored and owned by ‘Hot Sake LLC’. TMTS is a new system to be built & is not extended from any previously existing software. The design, development, testing, maintenance, and overall operations will be handled by ‘Team 2’, of the spring 2021 Software Engineering Capstone course of Kennesaw State University.

The system is developed using Flutter, a framework created by Google for building User Interfaces for mobile applications. Flutter utilizes the Dart programming language(also created by Google) to handle functionality. Firebase, similarly created by Google will be utilized for authentication & security, as well as database needs. Flutter was decided upon because of the ability to build applications for both iOS & Android with a single codebase.

Text Marks the Spot’s primary functionality is based on a sender & receiver type interaction between users. A user (‘sender’) is able to create a ‘Text Mark’ at a location of their choice. A ‘Text Mark’ is a message set by a sender at a specific location for a recipient (‘Receiver’) to view. A ‘receiver’ can only view the received ‘Text Mark’ when they are actively within the location of the Text Mark. Text Marks the Spot will utilize a map functionality to display both sent & received Text Marks for a user.

Splash/Sign-In Screen

This is the first screen that a user sees upon opening the app for the first time. The user can choose to log-in via their Google or Apple account.

Whichever option they choose will log them into the app and use that to recognize them in the future.

The app will automatically keep a user logged in so future uses will result in them being directed to the Home screen instead.

Home Screen

This is the Home screen that the user sees after being logged in and everytime they open the app after initial log-in.

It consists of 4 different buttons:

  • Create a Text Mark:
    User clicks on this button to pinpoint a new text mark location.
  • Settings:
    User clicks on this button (Gear icon) to go to Account Settings screen
  • GPS:
    User Clicks on the GPS icon to center the map to a new location they want to mark.
  • Text Marks:
    User clicks on the Text icon to go to the Text Marks screen to view their text marks received.


Create Text Mark Screen

This is the Create a Text Mark screen the user sees when adding a new text mark. The user fills out the required boxes and then hits the Send button to submit the text mark.

It consists of 4 different fields:

  • Nickname:
    This is the first box, it is a nickname for the place you've marked for the user. Gym is the nickname in this text mark.
  • Username:
    This is the user for the text mark to be sent to. "googleAustin" is the username in this example.
  • Message:
    This is the message to send to the recipient of the text mark. "20% discount..." is the message in this example.
  • Time:
    This is how long the text mark should stay active for. 30 days in this example.


Sent Text Marks

After clicking on the "Text Marks" icon in the main home screen, the user can view both received and sent text marks. The user can swipe right/left to see either sent or recieved.

Each textmark shown on the list is arranged in chronological order. Textmarks listed show date recieved, user sent/received, and the location nickname.

In this example, the user has navigated to the "Sent Textmarks" screen and has 1 textmark displaying on their list. The user can then click on the textmark to read it or they can select it in order to modify the text mark.

Delete Text Mark

After a user selects a text mark in the "sent" or "received" text marks sections then they can choose to delete the text mark. In this example the user has selected the "Salt and Pepper Cafe" text mark and chosen the "delete" option.

The app will prompt the user to confirm they want to delete the text mark as shown on the screen above. Once deleted the text mark is permanently gone from both the user's list of text marks and from the map and must be re-added in order to appear again.

Account Settings

This is the settings screen the user accesses when they want to modify their app settings.

It consists of 4 different options:

  • Change Username:
    User can change their display name on the app. In this example appleAustin would be changed.
  • Location Services:
    User can modify their GPS settings for the app as needed.
  • Help:
    Directs the user to a page with common help topics and a way to report app issues.
  • Logout:
    The user will logout of the app and will be re-directed to the Splash/Log-In screen
